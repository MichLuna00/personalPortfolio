Habilidades

Conocimientos fundamentales en lógica de programación, estructuras condicionales y repetitivas. Manejo básico de lenguajes como HTML, CSS, JavaScript y PHP. Capacidad para crear formularios, validar datos e interactuar con bases de datos utilizando MySQL. Familiarizado con entornos de desarrollo como XAMPP y el uso de editores de código como Visual Studio Code. Habilidad para resolver problemas simples y trabajar en pequeños y medianos proyectos y apps web.

HTML

Aprendí a estructurar páginas web utilizando etiquetas semánticas para encabezados, párrafos, listas, formularios, tablas e incrustación de contenido multimedia. Conozco cómo organizar correctamente el contenido para una mejor accesibilidad y compatibilidad entre navegadores.

CSS3

Adquirí conocimientos para aplicar estilos visuales a páginas web, manejar colores, fuentes, márgenes, paddings, bordes, fondos, animaciones y transiciones. También sé utilizar Flexbox y Grid para crear diseños responsivos y adaptables a diferentes tamaños de pantalla.

Javascript

Manejo básico de la lógica de programación en el navegador: uso de variables, funciones, condicionales, ciclos, arrays y objetos. Aprendí a manipular el DOM para crear interactividad en páginas web y a manejar eventos como clics, envíos de formularios y validaciones básicas.

React

Comprensión inicial de cómo crear interfaces dinámicas mediante componentes reutilizables. Aprendí a usar el estado (state) y las props, así como a organizar pequeñas aplicaciones de forma modular. Manejo básico de hooks como useState y useEffect.

MySql

Capacidad para diseñar y gestionar bases de datos relacionales. Aprendí a crear tablas, insertar, actualizar, eliminar y consultar datos usando sentencias SQL básicas. Comprensión del uso de claves primarias, foráneas y relaciones entre tablas.

Node JS

Introducción al desarrollo del backend usando JavaScript. Aprendí a crear servidores básicos, manejar rutas, trabajar con el módulo fs para leer y escribir archivos, y gestionar solicitudes HTTP. Uso inicial de Express.js para crear APIs simples.
